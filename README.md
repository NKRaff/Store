## About Laravel

Laravel is a web application framework with expressive, elegant syntax. We believe development must be an enjoyable and creative experience to be truly fulfilling. Laravel takes the pain out of development by easing common tasks used in many web projects, such as:

- [Simple, fast routing engine](https://laravel.com/docs/routing).
- [Powerful dependency injection container](https://laravel.com/docs/container).
- Multiple back-ends for [session](https://laravel.com/docs/session) and [cache](https://laravel.com/docs/cache) storage.
- Expressive, intuitive [database ORM](https://laravel.com/docs/eloquent).
- Database agnostic [schema migrations](https://laravel.com/docs/migrations).
- [Robust background job processing](https://laravel.com/docs/queues).
- [Real-time event broadcasting](https://laravel.com/docs/broadcasting).

Laravel is accessible, powerful, and provides tools required for large, robust applications.

## Modo de Instalação

Modo de Instalação

<strong>1-</strong> Baixe e instale o PHP pelo site oficial em https://www.php.net/downloads.php

<strong>2-</strong> Baixe e instale um SGBD (qualquer um de sua preferencia), em seguida crie um banco de dados

<strong>Recomendação</strong> - Baixe e instale o XAMPP, pois ele ja vem tanto com o PHP quanto com o MySQL

<strong>3-</strong> Baixe o Composer pelo site oficial em https://getcomposer.org/download/

<strong>4-</strong> Instale o Composer dentro da pasta do PHP

<strong>5-</strong> Clone o código que esta disponivel em https://github.com/NKRaff/Store

<strong>6-</strong> Na pasta do projeto, escreva no terminal composer install

<strong>7-</strong> Duplique o arquivo ".env.example", e renomei o duplicado para ".env"

<strong>8-</strong> No arquivo ".env", faça a configuração do banco de dados

<strong>9-</strong> Ainda no terminal escreva os comandos a segui
 - php artisan migrate
 - php artisan key:generate
 - php artisan serve

<strong>10-</strong> Na URL do seu browser, escreva "localhost:8000" ou "127.0.0.1:8000"

<strong>11-</strong> Crie um usuario comum

<strong>12-</strong> No banco de dados, na tabela users, altere o role_as do usuario para ADM

<strong>13-</strong> Deslogue o usuario no site

<strong>14-</strong> Pronto, o projeto ja esta pronto para uso
